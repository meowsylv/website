minesweeper.png is a recreation of minesweeper on Discord.

How to start a new game

  1. Generate a link.

    First you need to generate a brand new game link, you can do this by either going to https://www.papaproductions.cc/minesweeper.png and copying the link after it's done loading, or you can use this little tool I made.
    The generated link will show up here.

  2. Post the link

    Post the link anywhere on Discord. And you're done!

How to play

This game has two commands. In order to use them, you must send them as messages.

  1. s/$/dx,y:$: Opens up a square. Example: s/$/d1,1:$
  2. s/$/fx,y:$: Flags a square. Example: s/$/f2,3:$
